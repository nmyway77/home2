01 · OVERVIEW

유형별 출제 비중

시험 100점이 6개 묻는 방식별로 어떻게 분배되었는지 한눈에. 가장 큰 비중은 서답형/응용(38점) + 독해(15.9점) = 53.9점으로 시험의 절반 이상. 서답형 6문항이 38점이라는 점이 대일고 시험의 핵심 색채 — 단순 객관식 풀이가 아닌 본문 문장 정확 영작 능력이 점수를 결정. 어법(10.1점)·어휘(8.0점)·구조(13.3점)·빈칸(12.7점)이 비교적 고르게 분배되어 특정 유형 편중 학습은 비효율.

TRAP DETECTED · 글 구조 함정 패턴
지시어 + 연결어 + 인과 흐름 = 위치 결정
순서 배열은 모의고사 단일 단락을 (A)(B)(C)로 분할하여 재구성. 각 단락 시작의 지시어/연결어가 결정적 단서:
▸ Q5: "This is because"(C 시작 — 앞 결론의 이유) → "Judgments... are based on"(A 시작 — 새 화제) → "Both entities, therefore"(B 시작 — 결론)
▸ Q6: "That is because the room is not part"(C — 헤드폰 이유) → "That's because the room is now part"(B — now가 not과 대조) → "Similarly"(A — 유사 확장)

▸ 문장 삽입은 주어진 문장의 This/These/However/this need 같은 지시어가 받을 대상이 앞에 와야 함. Q9 'This is where' / Q10 'this need + these constraints' / Q11 'However + hope'.
▸ Q5·6 (각 2.9점) 순서 배열 · Q9 (2.0점)·Q10 (2.9점)·Q11 (2.6점) 문장 삽입 · 총 13.3점

08 · CONCLUSIONS

유형별 학습 전략

유형별 출제 패턴이 파악되면 학습 시간 배분도 달라져야 합니다. 대일고 시험은 서답형/응용(38점) + 독해(15.9점) = 53.9점이 핵심이며, 단순 객관식 풀이가 아닌 본문 영작 능력 + 다중 조건 처리 + 정밀 독해를 동시에 측정합니다.

CONCLUSION 01.
서답형 38점 — 본문 문장 통째 암기가 핵심
서답형 6문항이 시험의 38점(38%)을 차지하는 결정적 영역. 서4·서5·서6 정답은 본문/수특 문장 100% 동일(Procter "immersed herself in the exploration...", Kim "searched for a job related to art..."). 본문 문장 통째 암기가 가장 효율적. 학원 모의 서답 훈련으로 철자·인칭·시제·어순 4중 정밀도 확보.
CONCLUSION 02.
독해 15.9점은 무관문장 + paraphrase 식별
독해 6문항 중 무관문장 2개(Q7·8) 5.2점이 가장 큰 함정. 같은 소재로 창작된 문장이 흐름을 끊는 형태 — "단어가 같다고 흐름도 같다"는 착각이 함정. 요지·주제·제목(Q17·18·19)은 본문 핵심을 paraphrase한 보기 식별. 본문 핵심 어휘 → 동의어 매칭 훈련.
CONCLUSION 03.
글 구조 13.3점 — 지시어/연결어 단서 자동화
순서 배열·문장 삽입 5문항이 13.3점. 모의고사 단일 단락이 (A)(B)(C)로 분할되거나 한 문장이 분리됨. 결정적 단서: This/These/this need + However/therefore + 지시 부사(now/Similarly). 평소 학습 시 각 단락 첫 단어를 색상 표시하는 습관.
CONCLUSION 04.
빈칸 12.7점 — 본문 단어 동의어 매칭
빈칸 5문항 모두 본문 핵심 표현의 paraphrase가 정답. Q12 everyday→ordinary, Q13 limit→restrict, Q23 motivation→reason. 본문 정독 시 핵심 단어 옆에 동의어 메모 습관. 빈칸 직전·직후 문장이 결정 단서(Q21 'even support').
CONCLUSION 05.
어법 10.1점 — 4가지 다른 영역 균형 학습
4문항이 모두 다른 어법 영역. Q14 시제일치 / Q15 분사+태 / Q16 관계사 / Q24 관계부사+동명사+등위접속사. 한 영역 편중 학습은 위험. 매주 어법 5개 영역 균등 1문항씩 풀이 + 학교 모의고사 어법 함정 패턴 분석. "옳은 것 개수"(Q15)는 함정 모두 식별 필요.
CONCLUSION 06.
어휘 8점은 외부 어휘책으로만 대비
어휘 4문항(8점)은 시험범위 자료에 단 1개도 등장하지 않는 외부 출처. 본문 학습으로는 절대 대비 불가. 학년별 어휘책(능률·천일문 등) + 내신 빈출 어휘 정리 노트 필수. 헷갈리는 형태(obvious↔obscure / alerted↔altered / venue↔revenue)는 별도 카드로 묶어 암기. 92→100점 도전의 마지막 관문.
