Three definitive wrong answers. 시험지 흔적이 명확히 보여주는 오답 3개

시험지에 남은 학생의 풀이 흔적과 정답을 비교하여 명확하게 오답이 확인된 세 문항을 분석합니다. 이 세 문항만으로 −11점이 손실되었으며, 모두 "꼼꼼히 끝까지 읽고 모든 조건을 확인하지 못하는" 동일 패턴입니다.

Q24
분사 + 수일치 빈칸 (A)(B)(C)
학교 프린트 어법 · 다중 어법 포인트 동시 적용
3점
(A) working / (B) designed 분사 능동·수동은 정확히 골랐으나, (C) clear criteria that [is / are] useful에서 "criteria"가 복수형(criterion의 복수)임을 놓치고 'is'를 선택했습니다. 목운중 시험에서 자주 출제되는 "수일치" 함정에 걸린 전형적인 사례입니다. 학교 프린트 4대 문법 중 "수일치"까지는 학습이 도달하지 못한 상태입니다.
유하담 학생 답
⑤ working / designed / is

(A) working ✓ (B) designed ✓ (C) is ✗

→ 분사는 맞췄으나 수일치 누락. criteria를 단수로 착각.

정답
② working / designed / are

(A) working ✓ (B) designed ✓ (C) are ✓

→ criteria = criterion(기준)의 복수형. 동사도 복수 are.

Q16
어법상 옳은 문장의 개수는?
교과서 본문 L1 · 다중 정답 개수 세기 · 4점 고배점
4점
본문 ⓐ-ⓔ 5개 문장 중 어법상 옳은 문장의 개수를 묻는 문제. 옳은 문장은 ⓐ, ⓑ, ⓓ 세 개이지만, 유하담 학생은 두 개로 잘못 셈. 시험지에 ⓐⓑⓒⓓⓔ에 본인 판단을 메모했으나 ⓓ "I have been using Spanish a lot in my daily life" (현재완료 진행 — 옳음)를 틀렸다고 잘못 판단한 흔적이 있습니다.
유하담 학생 답
② 2개

학생 판단: ⓐ ✓, ⓑ ✓, ⓓ ?
→ 결론: 2개로 셈

→ 현재완료 진행형 "have been using"을 옳지 않다고 판단

정답
③ 3개

옳은 문장: ⓐ ✓, ⓑ ✓, ⓓ ✓ (3개)
틀린 문장: ⓒ (which→what), ⓔ (using→used)

→ 현재완료 진행형은 어법상 옳은 표현

−6점 추정

−7점 추정
