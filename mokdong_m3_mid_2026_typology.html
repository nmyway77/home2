01 · OVERVIEW

유형별 출제 비중

08 · SPECIAL PATTERNS

목동중 시험의 5가지 특이 출제 패턴

유형 분류와 별도로, 목동중 시험에서 반복적으로 등장하는 구조적 함정 패턴을 추출. 이 패턴들은 유형(어법/독해/문맥…)을 가로질러 나타나며, 한 문항에 둘 이상 결합되기도 함.

2
PATTERN 01 · 8점
"정답 2개" 형식 — 부분 점수 없음
정답을 둘 다 골라야만 인정. 한 개만 골라서는 완전 오답. 이 형식의 함정은 "명백한 한 개" + "찾기 어려운 한 개"로 구성되어, 쉬운 답에서 멈추면 0점.
▸ Q6 다의어 쓰임 (4점) · Q22 결론 빈칸 부적절 (4점)
↺
PATTERN 02 · 7점
"안 들어가는 것" 역추론
보기 5개 중 빈칸에 들어갈 수 없는 단어. 보기 5개를 모두 빈칸에 대입해 검증해야 답이 나오는 노동 집약형. 시간 안배가 결정적 — 빠른 매칭 학습 필요.
▸ Q15 단어 빈칸 (3점) · Q19 전치사 빈칸 (4점)
⊞
PATTERN 03 · 12점
시각화 자료 변환 (표·흐름도·절차)
본문 → 표·흐름도·절차도로 시각화한 뒤 문제 출제. 본문 문장을 정보 단위로 분해해 시각 자료에 매핑하는 능력 측정. 단순 일치보다 한 단계 위 인지 작업.
▸ Q12 T/F 표 · Q23 흐름도 · Q26 절차도
≡
PATTERN 04 · 21점
5개 보기 동시 검증형
"5개 중 옳은 것" 또는 "5개 중 어색한 것 1개" 형태. 5개 모두를 본문·문법 규칙에 정밀 대조해야 풀 수 있음. "보기마다 O/X/△ 표시 → △ 재검토" 풀이 절차 훈련 필수.
▸ Q10 어법 (3점) · Q11 카운트 (4점) · Q13 어법 (3점) · Q17 어법 (4점) · Q20 어색한 문장 (4점) · Q24 사역동사 (4점)
↔
PATTERN 05 · 12점
인과 방향·반의어 함정
본문의 원인-결과 자리를 뒤집거나, 동사·형용사를 반의어로 바꿔치기한 함정. 본문을 의미 단위까지 분해해 읽어야 식별 가능. 단순 단어 매칭으로는 함정에 걸림.
▸ Q22 결론 인과 방향 (4점) · Q27 Eloise 인과 뒤집기 (4점) · Q12 시제·사실 (4점)
+
PATTERN 06 · 12점
다중 조건·다중 규칙 결합
한 문항에 두 가지 이상 조건/규칙을 동시 적용해야 답이 나오는 형식. Q25는 어법 + 본인 작업 이중 조건, Q28은 명사+to부정사 + 전치사 + -thing 어순 3중 규칙. 한 조건만 보면 함정 답으로 빠짐.
▸ Q24·25 사역동사 + 본인 작업 (8점) · Q28 to부정사 다중 규칙 (4점)
09 · CONCLUSIONS

유형별 학습 전략

유형별 출제 패턴이 파악되면 학습 시간 배분도 달라져야 합니다. 목동중 시험은 어법(29점) + 독해(25점) = 54%가 핵심이며, 단순 암기보다 다중 처리·시각화·역추론 등 메타 인지 능력 측정에 집중되어 있습니다.

CONCLUSION 01.
어법은 5대 영역 모두 출제 — 학교 프린트 학습지 핵심
29점 8문항이 사역동사(8점) · if/whether·간접의문문(7점) · 관계대명사(7점) · to부정사(4점) · 분사(3점) 5대 영역에 골고루 분포. 학교 프린트 if/whether 학습지·to부정사 학습지·사역동사 학습지의 패턴이 거의 그대로 출제. 학습지 패턴 + 예문 암기가 어법 영역 만점 직결.
CONCLUSION 02.
독해는 일치/불일치가 압도 — 본문 정밀 독해
독해 25점 7문항 중 일치/불일치 5문항(18점). 추론형(요지)은 1문항(3점)뿐. 다양한 형태(인터뷰·T/F 표·흐름도·진술)지만 본질은 모두 "본문 문장 vs 선지 문장의 정밀 매칭". 본문 한줄해석 자료 5회독이 가장 직접적 효과.
CONCLUSION 03.
"5개 보기 동시 검증" 21점 — 풀이 절차 훈련
시험에서 가장 빈도 높은 함정 패턴. 6문항 21점이 5개 보기를 모두 검증해야 답이 나오는 형식. 평소 학습에서 "보기마다 O/X/△ 명확히 표시 → △는 마지막에 재검토" 풀이 절차를 절대 절차화. 한 보기만 잘못 판단해도 다중 정답 형식(Q11·Q24·Q28)은 통째로 오답.
CONCLUSION 04.
시각화 자료 12점 — 본문 → 표·흐름도 변환 훈련
T/F 표(Q12)·흐름도(Q23)·절차도(Q26) 형태로 3문항 12점이 본문 정보의 시각화 변환을 요구. 평소 학습 시 본문 단락별 핵심 정보를 표·도식으로 직접 그려보는 능동적 정리 작업이 효과적. 학교 한줄해석 자료의 표·흐름도 정리 부분(예: 컴포트 푸드 4종↔성격) 정밀 학습.
CONCLUSION 05.
"안 들어가는 것" 역추론 7점 — 다중 매칭 속도
Q15·Q19 두 문항(7점)이 보기 5개를 모두 빈칸에 대입해야 답이 나오는 노동 집약형. "빈칸 위치 → 보기 의미 → 자연스러움 판별"을 빠르게 반복. 학교 프린트 어휘 짝(be similar to / be filled with / think of / make A for B) 자동 호출 수준까지 연습.
CONCLUSION 06.
응용/추론 8점은 사고력 — 학습 외 영역
Q25 용돈 계산·Q26 절차 역추적은 학습으로만 풀기 어려운 다중 조건 추론. Q25는 어법 + 본인 작업 이중 조건, Q26은 6단계 + 4경고문 + 인물 상황 3단계 체인. "조건 모두 만족하는 경우만 인정" 사고 절차 훈련. 90점대 안정 득점이 목표라면 이 8점은 아껴두고 시간 마지막에 정밀 분석하는 전략 추천.
CONCLUSION 07.
변형도 분포가 학습 우선순위 결정 — 미세 54점 + 소폭 34점
28문항을 원지문 변형 강도로 보면 미세 변형 16문항(54점)이 학습 직결 영역. 본문·학습지 100% 암기 시 즉시 풀이 가능. 소폭 변형 9문항(34점)은 학교 패턴(사역동사·if/whether 등) + 새 시나리오 결합형으로 패턴 + 응용력 동시 필요. 대폭 변형(8점) + 신규 창작(4점) = 12점은 본문 정밀 분석 + 사고력 영역. 학습 우선순위는 미세 → 소폭 → 대폭 → 신규 순. 미세+소폭 88점에 학습 시간의 80%를 배분하는 것이 가성비 최적.
