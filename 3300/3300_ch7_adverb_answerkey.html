Unit 14 · 부사 찾기+용법
다음 문장에서 부사를 찾아 밑줄을 긋고, <보기>에서 용법을 골라 쓰세요. <보기> 동사 수식 · 형용사 수식 · 부사 수식 · 문장 전체 수식
1.Judy speaks Chinese fluently.
2.Thank you so much.
3.I'm too shy to talk to her.
4.Fortunately, we won the game.
정답 & 해설 보기
Answer · 부사 + 용법
1.fluently — 동사(speaks) 수식 → 동사 수식
2.much — 동사(구) 수식 → 동사 수식 / so — 부사(much) 수식 → 부사 수식
3.too — 형용사(shy) 수식 → 형용사 수식
4.Fortunately — 문장 전체 수식 → 문장 전체 수식
📌 POINT부사는 동사·형용사·부사·문장 전체를 수식한다. 문장 앞의 -ly 부사(Fortunately, Happily 등)는 대개 문장 전체를 수식한다.

Unit 14 · 빈칸 채우기
빈칸에 알맞은 의문부사나 관계부사 또는 복합관계부사를 쓰세요.
1. long have you studied history?
2.This is the spot car accidents occur very frequently.
3. did you leave me? – Because I thought you loved Jason.
4.I will tell you I have succeeded.
5.I always make a lot of mistakes I make a composition.
6. difficult it may be, get up the courage and do it.
7.Take care of yourself you go.
8.Do you know the time the game starts?
정답 & 해설 보기
Answer · 의문/관계/복합관계부사
1.How — 「얼마나 오래」 의문부사(정도)
2.where — 선행사 the spot(장소) → 관계부사 where
3.Why — 이유를 묻는 의문부사(답이 Because)
4.how — 「어떻게 ~했는지」 방법 (관계부사 how = the way)
5.when — 「~할 때마다」 시간 관계부사/접속
6.However — 「아무리 ~할지라도」 복합관계부사(양보)
7.wherever — 「어디를 가든」 복합관계부사(양보/장소)
8.when — 선행사 the time(시간) → 관계부사 when
📌 POINT선행사가 시간이면 when, 장소면 where, 이유면 why, 방법이면 how. 복합관계부사 However는 뒤에 형/부가 바로 온다: However + 형용사/부사 + S + V.

Unit 14 · 어법 선택
다음 문장에서 어법상 맞는 어구를 고르세요.
1.I tried ( hard / hardly ) to win the first prize.
2.You two ( hard / hardly ) know each other.
3.Have you seen her ( late / lately )?
4.I ( near / nearly ) fell down at the news.
5.Everyone spoke ( high / highly ) of him.
정답 & 해설 보기
Answer · 어법 선택
1.hard — 열심히 노력했다 → hard(열심히). hardly는 「거의 ~않다」
2.hardly — 거의 서로 모른다 → hardly
3.lately — 최근에 → lately. late는 「늦은/늦게」
4.nearly — 거의 넘어질 뻔 → nearly. near는 「가까운/가까이」
5.highly — speak highly of ~ 「~을 높이 평가하다」
📌 POINT형태가 같지만 -ly가 붙으면 뜻이 달라지는 부사: hard(열심히)/hardly(거의 ~않다), late(늦게)/lately(최근에), near(가까이)/nearly(거의), high(높이)/highly(매우, 높이 평가하여).

Unit 15 · ago/before/since
다음 문장에서 어법상 알맞은 단어를 고르세요.
1.The shop closed an hour ( ago / before / since ).
2.Have you met her ( ago / before / since )?
3.They got married a week ( ago / before / since ).
4.Jeff suddenly disappeared last year and he hasn't shown up ( ago / before / since ).
5.I visited Jeju-do ( ago / before / since ).
6.He said he had seen it three days ( ago / before / since ).
7.I have forgotten what you said to me ( ago / before / since ).
8.He left for the Sahara Desert and hasn't been seen ( ago / before / since ).
9.Long, long ( ago / before / since ), there lived a magician in the forest.
10.Haven't we met ( ago / before / since ), Adolf?
정답 & 해설 보기
Answer · 어법 선택
1.ago — 현재 기준 「~전에」 + 과거시제 → ago
2.before — 현재완료 + 기간 없이 「이전에」 → before
3.ago — 현재 기준 一주 전 + 과거 → ago
4.since — 과거 특정 시점 이후로 + 현재완료 → since
5.before — 기간 표현 없이 「이전에」 경험 → before
6.before — 과거완료(had seen) 기준 「3일 전에」 → before
7.before — 현재완료 + 기간 없이 「이전에」 → before
8.since — 과거 시점 이후로 + 현재완료 → since
9.ago — 옛날 옛적에(Long long ago) 관용 → ago
10.before — 현재완료 + 기간 없이 「이전에」 → before
📌 POINTago=현재 기준 「~전」(항상 과거시제). before=과거 기준 「~전」(주로 과거완료)·기간 표현 없이 단독으로 현재완료/과거와 함께. since=과거 특정 시점 「~이후로」(현재완료/과거완료).

2.ever — 현재완료 의문문 경험 「지금까지」 → ever

Unit 15 · 타동사+부사
다음 문장에서 어법상 올바른 표현을 고르세요.
1.He ( handed in the report / handed in it ) yesterday.
2.Don't ( give up it / give it up ).
3.My daughter ( put the raincoat on / put on it ).
4.Her sister ( cares sick people for / cares for sick people ) sincerely.
5.I ( agreed with him / agreed him with ) on the issue.
6.We can't ( dispense with air / dispense air with ).
7.Please ( take your hats off / take off them ) in this room.
8.They ( turned them on / turned on them ) in the evening.
9.We ( live the rice on / live on the rice ) in Asia.
10.( Fill it up / Fill up it ), please.
11.Have you ( figured out how to cook doenjangjjigae / figured out it )?
12.( Pick up the phone / Pick up it ) now.
정답 & 해설 보기
Answer · 어법 선택
1.handed in the report — 목적어가 명사면 어순 자유
2.give it up — 목적어가 대명사(it)면 「동사+목적어+부사」
3.put the raincoat on — 명사 목적어는 부사 앞/뒤 자유(on it는 불가)
4.cares for sick people — care for는 「자동사+전치사」 → 목적어는 전치사 뒤
5.agreed with him — agree with = 자동사+전치사
6.dispense with air — dispense with = 자동사+전치사
7.take your hats off — them(대명사)은 부사 앞에만 → take them off가 정답 어순, take off them 불가
8.turned them on — 대명사 목적어 → 동사+대명사+부사
9.live on the rice — live on = 자동사+전치사 「~을 주식으로 하다」
10.Fill it up — 대명사 목적어 → 동사+it+부사
11.figured out how to cook doenjangjjigae — 명사구 목적어 가능(it은 out 뒤 불가)
12.Pick up the phone — 명사는 부사 뒤 가능, 대명사 it은 불가
📌 POINT「타동사+부사」의 목적어가 대명사일 때는 반드시 「동사+대명사+부사」(turn it on ○ / turn on it ✕). 명사일 때는 앞뒤 모두 가능. 「자동사+전치사」(agree with, care for, live on, dispense with)는 목적어가 전치사 뒤에 온다.

다음 문장에서 어법상 알맞은 어구를 고르세요.
1.What did you say to her ( just / just now )?
2.Jane ( just / just now ) arrived in Seoul the day before.
3.The boy has been very ill, but he's much better ( just / now ).
4.He is very busy ( just / just now ), so you'll have to wait.
5.I have ( just / just now ) done it.
6.I was ( just / now ) leaving the office when she came to meet me.
7.I'm hungry ( just / just now ).
8.( Now / Once ), let's make a plan for the project.
정답 & 해설 보기
Answer · 어법 선택
1.just now — 과거형과 함께 「방금」
2.just — 「막」의 뜻으로 과거 서술 (the day before와 호응)
3.now — 「지금은 훨씬 나은」
4.just now — 현재 「바로 지금」 매우 바쁘다
5.just — 현재완료 + just 「막」
6.just — 과거진행 + just 「막 ~하려던 참」
7.just now — 현재 「바로 지금」 배고프다
8.Now — 「자, 이제」 문두
📌 POINTjust=「막」(현재완료·과거진행), 「곧」(미래). just now=「방금」(과거형), 「바로 지금」(현재진행). now=「지금」(현재).

Unit 15 · 배열(so/neither)
괄호 안의 단어를 바르게 배열하여 문장을 완성하세요.
1.I hate to play basketball. – ( Jane / so / does )
2.My kids went on a picnic. – ( so / my wife / did )
3.I can't tell you the truth. – ( can / she / nor )
4.I must clean the classroom. – ( must / so / we )
5.He isn't studying now. – ( I / am / neither )
6.I don't like swimming. – ( he / neither / does )
7.I'll never sell this house to her. – ( I / neither / will )
8.He seldom handed in his reports. – ( she / nor / did )
9.My mother was beautiful in her youth. – ( was / so / my mother )
10.I am not a coward. – ( I / am / nor )
정답 & 해설 보기
Answer · 배열
1.→ So does Jane. (긍정 동감 So+do+S)
2.→ So did my wife. (긍정 동감(과거) So+did+S)
3.→ Nor can she. (부정 동감 Nor+can+S)
4.→ So must we. (긍정 동감(조동사) So+must+S)
5.→ Neither am I. (부정 동감 Neither+be+S)
6.→ Neither does he. (부정 동감 Neither+do+S)
7.→ Neither will I. (부정(never) 동감 Neither+will+S)
8.→ Nor did she. (부정(seldom) 동감 Nor+did+S)
9.→ So was my mother. (긍정 동감(be동사) So+was+S)
10.→ Nor am I. (부정 동감 Nor+be+S)
📌 POINT긍정에 동감: So+(be/조동사/do)+주어 「~도 그렇다」. 부정에 동감: Neither/Nor+(be/조동사/do)+주어 「~도 그렇지 않다」. never·seldom도 부정으로 취급.
H-2
Unit 15 · hardly/seldom
다음 문장에서 어법에 맞는 단어를 고르세요.
1.We had ( hardly / seldom ) gone a mile when we came across Jane.
2.Mother ( hardly / rarely ) washes the dishes lately because she's tired.
3.He could ( scarcely / rarely ) walk because of the accident.
4.I'll try ( hard / hardly ) to forget your name.
5.I can ( hardly / seldom ) believe that I had a date with her.
6.He ( hardly / rarely ) eats hamburgers.
7.( Hardly / Rarely ) had he seen me when he ran up to me.
8.( Scarcely / Seldom ) had he seen me before he ran up to me.
정답 & 해설 보기
Answer · 어법 선택
1.hardly — hardly ~ when 「~하자마자」 정도
2.rarely — 「좀처럼 ~않다」 빈도
3.scarcely — 「거의 ~못하다」 정도
4.hard — 「열심히」 → hard
5.hardly — 「거의 ~못하다」 정도
6.rarely — 「좀처럼 ~않다」 빈도
7.Hardly — Hardly ~ when 도치
8.Scarcely — Scarcely ~ before 도치
📌 POINThardly·scarcely=(정도) 「거의 ~않다」. seldom·rarely=(빈도) 「좀처럼 ~않다」. Hardly/Scarcely ~ when/before 「~하자마자」 구문은 도치(had+S)를 동반한다.
