03.Seven Principles of Mokun

Seven principles behind every item. 목운중 고유 7대 출제 원칙

30문항을 샅샅이 분석하여 도출한 목운중의 출제 원칙 7가지입니다. 이 원칙들을 이해하면 앞으로의 기말·2차 중간고사 방향을 예측할 수 있습니다.

Principle 01.
서답형 대신 "선택형 고난도"로 사고력 측정
서답형 0문항인 대신, "배열의 ★위치 고르기"(2번·27번), "어법 틀린 개수 세기"(16·21번), "일치 개수 세기"(7·29번) 형태로 서답형 수준의 사고력을 선택형에 녹여 냅니다. 단순 5지선다로 보이지만 실제 풀이 과정은 서답형 영작·분석과 동일.
Principle 02.
외부지문을 세트형으로 묶어 출제
한 외부지문에 2~4문항을 묶어 출제 — SR1 4문항 / SR2 3문항 / SR3 2문항. 외부지문 한 개를 놓치면 최대 4문항(14점)을 잃습니다. 세트형 문항들은 같은 지문 안에서 문장 삽입·어법·어휘·연결사를 모두 요구하므로 원본 숙지가 필수.
Principle 03.
프린트 자료를 독립 출제 풀로 활용
75개 동사 숙어 → 3문항(5·9·23번) / 영영풀이 → 2문항(11·29번) / 문법 포인트 → 4문항. 프린트 자료가 교과서와 동등한 지위를 갖습니다. 프린트를 숙지하지 못한 학생은 9문항(30점)을 그대로 잃습니다. 학원 수업에서 제공하는 프린트 완벽 암기가 변별력의 첫 관문.
Principle 04.
원본에 없는 정보를 의도적으로 추가
L2 28번에 "비타민 C" 정보 신규 추가, L2 30번에 "86억 뉴런 · 표면적" 추가, SR3에 "이유 문장" 추가, L1에 "digital tools" 프레임 추가. 추가된 정보가 정답 선지의 근거가 됩니다. 원본만 암기한 학생은 오히려 혼란.
Principle 05.
반의어 치환을 내용 일치까지 확장
7번(hardly↔significantly), 10번(appropriate↔inappropriate), 28번(contain↔get rid of / many↔scarcely / improve↔damage), 30번(intricate↔simple) — 지문의 핵심 서술을 정반대로 뒤집은 선지를 함정으로 배치. 어휘 문제뿐 아니라 내용 일치 문제에서도 반의어 전략을 사용.
Principle 06.
교과서 본문을 대화문으로 각색
21번 최고난도 5점 문항은 Lesson 2 본문(Beneficial Foods)을 A-B 대화문으로 완전히 각색. 내용은 같지만 형식이 달라 원본 암기만으로는 대응 불가. 각색 후 5곳에 다른 종류의 어법 오류(전치사 누락, Each 단수, as~as 구문, healthy/health 형용사명사 등)를 심어 고난도화.
Principle 07.
한 문항에 다중 함정 심기 (고배점 구조)
21번(5점): 5개 다른 종류 함정 — 전치사 누락 + 형용사/명사 혼동 + Each 단수 + 부정사 오류 + as~as 구문 / 12번(4점): 수동태 + 계속적 용법 동시 / 16번(4점): 관계대명사 what/which + 분사 능동/수동 + 현재완료 진행 동시. 고배점 문항일수록 함정 개수가 증가하며, 하나의 문법 포인트가 아니라 여러 포인트를 동시에 체크하는 종합 문항으로 설계됨. 상위권 변별은 여기서 결정됩니다.
■ 학원 대비 전략 우선순위
필수 대비 영역 (비중 순)
  1. 외부지문 SR1~3 완벽 암기 — 9문항 28점. 여기서 놓치면 상위권 불가.
  2. 프린트 75 동사 숙어 — 3문항 직접 + 9번 선지 간접.
  3. 프린트 영영풀이 정확한 대응 — 11·29번 + 어휘 감각.
  4. 4대 문법 — 현재완료진행 / what / 계속적 용법(that·what 불가) / 분사 능동수동.
  5. 교과서 대화문 + 본문 구조 이해 — 단순 암기 X, 각색·프레임 변환까지.

damage ↔ improve / helpful
get rid of ↔ contain
scarcely ↔ many / some

appropriate ↔ inappropriate
create jobs ↔ deprive of jobs
intricate pattern ↔ simple pattern
