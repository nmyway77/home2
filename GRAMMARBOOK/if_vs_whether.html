English Grammar / Noun Clauses

~인지 아닌지를 묻는 두 단어 if vs. whether

둘 다 "~인지 아닌지"로 해석되어 서로 바꿔 쓸 수 있다고 배운다. 하지만 실제로는 whether만 쓸 수 있는 자리가 있고, 반대로 if는 조건의 의미("만약 ~한다면")로도 쓰여서 헷갈린다. 모든 차이를 한 번에 정리한다.

whether는 명사절 어디든 가능 — 만능.
if는 동사의 목적어 자리에서만 명사절로 쓸 수 있다.

그리고 if는 같은 자리에 와도 "만약 ~한다면"(조건)이라는 전혀 다른 의미로 쓰일 수 있어서 더 까다롭다.

01 / STRUCTURE

기본 구조 — 간접의문문의 어순

whether/if가 이끄는 명사절은 간접의문문이다. 일반 의문문과 어순이 다르다는 점을 꼭 기억해야 한다. 의문문은 동사가 주어 앞에 오지만, 명사절은 평서문 어순(주어 + 동사)을 따른다.

S + V + whether/if + S' + V'

✓ I don't know whether he will come.

그가 올지 안 올지 모르겠다.

✓ She asked me if I was free on Saturday.

그녀는 내가 토요일에 한가한지 물었다.

✕ 자주 하는 실수

✗ I don't know whether will he come.

✓ I don't know whether he will come.

간접의문문이므로 평서문 어순(주어+동사)을 따라야 한다. 일반 의문문 어순(will he come)은 틀린 표현이다.
02 / WHETHER ONLY
규칙 ②

whether만 쓸 수 있는 5가지 자리

IF CANNOT BE USED HERE — 시험 1순위!

다음 다섯 자리에서는 반드시 whether만 써야 한다. if는 절대 사용 불가이다. 시험에 정말 자주 출제되는 함정이니 꼭 외워둬야 한다.

❶ 주어 자리에 올 때

✓ Whether he agrees or not is not important.

✗ If he agrees or not is not important.

그가 동의하든 안 하든 중요하지 않다.

이유: if가 주어 자리에 오면 조건절("만약 ~한다면")로 잘못 해석될 수 있어서 헷갈림을 피하기 위해 whether만 허용.
❷ 보어 자리에 올 때

✓ The question is whether we can finish on time.

✗ The question is if we can finish on time.

문제는 우리가 제시간에 끝낼 수 있느냐다.

이유: be동사 뒤 보어 자리에 오는 명사절은 if 사용 불가. 주로 The question is ~, The issue is ~, My concern is ~ 같은 패턴.
❸ 전치사 뒤에 올 때

✓ It depends on whether you study hard.

✗ It depends on if you study hard.

네가 열심히 공부하느냐에 달려 있다.

✓ We talked about whether to cancel the trip.

우리는 여행을 취소할지에 대해 이야기했다.

✓ I'm not sure of whether it is true.

그게 사실인지 확신할 수 없다.

주요 전치사: on, about, of, over, as to 등. 전치사 + if 조합은 절대 안 됨 — 무조건 whether.
❹ whether or not (바로 붙여서) 쓸 때

✓ I wonder whether or not she likes me.

✗ I wonder if or not she likes me.

그녀가 나를 좋아하는지 안 좋아하는지 궁금하다.

주의: "or not"이 바로 뒤에 붙으면 whether만 가능. 단, 떨어져 있으면 OK → "I wonder if she likes me or not"는 자연스러움.
❺ 뒤에 to부정사가 올 때

✓ I can't decide whether to go or not.

✗ I can't decide if to go or not.

갈지 말지 결정을 못 하겠다.

✓ We discussed whether to accept the offer.

우리는 그 제안을 받아들일지에 대해 논의했다.

참고: "whether to V"는 "whether + 주어 + should + V"로 풀어 쓸 수 있다. (whether to go = whether I should go)
기억법 — 「주·보·전·or not·to V」

주어 / 보어 / 전치사 뒤 / or not 바로 뒤 / to부정사 앞 — 다섯 자리는 무조건 whether

이 다섯 자리만 외워두면 if/whether 시험 문제는 99% 잡힌다.

03 / IF'S TWO FACES
규칙 ③

if의 두 얼굴 — 명사절 vs 조건 부사절

SAME WORD, DIFFERENT MEANING

if는 두 가지로 쓰인다: "~인지 아닌지"(명사절)와 "만약 ~한다면"(조건 부사절). 한국어 해석과 문장 구조를 보면 구분할 수 있다.

Noun Clause
명사절 if
"~인지 아닌지"

I don't know if it will rain.

비가 올지 안 올지 모르겠다.

✓ whether로 대체 가능
Adverbial Clause
조건 부사절 if
"만약 ~한다면"

If it rains, I will stay home.

비가 오면, 집에 있을 거다.

✗ whether로 대체 불가
빠른 판별법

whether로 바꿔서 말이 되면 → 명사절 ("~인지 아닌지")
whether로 바꿔서 말이 안 되면 → 조건 부사절 ("만약 ~한다면")

04 / FUTURE TENSE

부사절 if/whether에서는 will 사용 불가!

if / whether가 조건 부사절("만약 ~한다면")이나 양보 부사절("~이든 아니든")로 쓰일 때, 미래의 일이라도 will 대신 현재 시제를 써야 한다.

A 조건 if (만약 ~한다면)

✗ If it will rain tomorrow, I will stay home.

✓ If it rains tomorrow, I will stay home.

내일 비가 오면, 집에 있을 거다.

B 양보 whether (~이든 아니든)

✗ Whether it will rain or not, we will go camping.

✓ Whether it rains or not, we will go camping.

비가 오든 안 오든, 우리는 캠핑을 갈 것이다.

⚡ 가장 헷갈리는 비교

✓ I don't know if it will rain tomorrow.

(명사절 → will 사용 가능)

✗ If it will rain tomorrow, I will stay home.

(부사절 → will 사용 불가)

같은 if라도 명사절이면 will 가능, 부사절이면 will 불가!

PRACTICE

실전 분석 — 규칙을 적용해보자

Test 1

✓ ___ he is honest is the most important thing. (If / Whether)

분석: 빈칸이 주어 자리! → 정답: Whether
Test 2

✓ I'm worried about ___ I'll pass the exam. (if / whether)

분석: 빈칸 앞이 전치사 about! → 정답: whether
Test 3

✓ She asked ___ I had finished the work. (if / whether)

분석: asked의 목적어 자리 (5가지 자리 아님) → if / whether 둘 다 가능
Test 4

✓ If she ___ tomorrow, I will be happy. (comes / will come)

분석: "내일 그녀가 오면 행복할 거다" → 조건 부사절 → will 불가 → 정답: comes
Test 5

✓ I don't know if she ___ tomorrow. (comes / will come)

분석: "그녀가 내일 올지 모르겠다" → 명사절(know의 목적어) → will 가능 → 정답: will come
Test 6

✓ We're still deciding ___ to leave now. (if / whether)

분석: 뒤에 to부정사가 옴! → 정답: whether
/ KEY TAKEAWAY

모든 게 두 줄로 정리된다

whether는 명사절에서 만능 —
주어, 보어, 전치사 뒤, or not 직전, to부정사 앞에서는 무조건 whether.

if는 동사의 목적어 자리에서만 명사절 OK —
그 외 자리에서는 조건의 if("만약 ~한다면")로 해석된다.

그리고 부사절의 if/whether에서는 절대 will을 쓰지 않는다. 명사절의 if/whether에서는 will을 써도 OK. 같은 단어, 다른 역할 — 영문법의 묘미다.
