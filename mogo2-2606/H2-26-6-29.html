①
01
The renowned British philosopher Alfred North Whitehead asserted that "civilization advances by extending the number of operations we can perform without thinking about them."
🔊
저명한 영국 철학자 Alfred North Whitehead는 "문명은 우리가 그것들(= 작업들)에 대해 생각하지 않고도 수행할 수 있는 작업의 수를 늘림으로써 발전한다"고 주장했다.
📌renowned저명한, 유명한assert that~라고 주장하다civilization문명extend늘리다, 확장하다operation작업, 조작perform수행하다
✏️ 문법assert that + 절: ~라고 주장하다. / by + 동명사(수단): by extending the number of operations = 작업의 수를 늘림으로써. / 목적격 관계대명사 생략: operations (that) we can perform = 우리가 수행할 수 있는 작업. / them = operations(작업들).
EASY 쉬운 풀이 【핵심 인용 — 글의 전제】
Whitehead: 문명은 "생각하지 않고도 할 수 있는 일"의 수를 늘려 발전한다. 즉 '자동화·습관화'가 곧 진보라는 것 — 이 글의 출발 명제. ★ 뒤의 '쿠폰' 예가 이 명제를 구체화한다.
02
Take, for example, the "advance" offered to civilization by the discount coupon, which allows consumers to assume that they will receive a reduced purchase price by presenting the coupon.
🔊
예를 들어, 할인 쿠폰이 문명에 제공한 '발전'을 생각해 보라. 그 쿠폰은 소비자가 쿠폰을 제시함으로써 그들(= 소비자)이 할인된 구매 가격을 받게 되리라고 가정하도록 해 준다.
📌offered to~에 제공된discount coupon할인 쿠폰allow O to VO가 ~하게 하다assume가정하다, 추정하다reduced purchase price할인된 구매 가격present제시하다
✏️ 문법과거분사 수식 offered: the "advance" offered to civilization = 문명에 제공된 '발전'(수동). / 계속적 용법 which: ..., which allows ... — 앞 the discount coupon을 부연(콤마 + which). / allow O to V: O가 ~하게 해 주다. / by presenting: 제시함으로써. they = consumers.
EASY 쉬운 풀이 【예시 — 할인 쿠폰】
쿠폰을 내밀면 값이 깎인다 — 소비자는 그냥 그렇게 될 거라고 자동으로 가정한다. '생각 없이 작동'하는 대표 사례. ★ 쿠폰 = "생각하지 않아도 되는 작업"의 구체적 예.
②
03
The extent to which we have learned to operate mechanically on that assumption is illustrated in the experience of one automobile-tire company.
🔊
우리가 그 가정에 따라 기계적으로 작동하는(반응하는) 법을 배운 정도는 한 자동차 타이어 회사의 경험에서 잘 드러난다.
📌the extent to which~한 정도operate mechanically기계적으로 작동·반응하다assumption가정, 추정be illustrated잘 드러나다, 예시되다
✏️ 문법전치사 + 관계대명사 to which: The extent to which we have learned ... = 우리가 ~을 배운 정도(to the extent의 to가 관계사 앞으로). 뒤는 완전한 절. / 수동태 is illustrated: (정도가) 잘 드러나다·예시되다. / 주어 핵 The extent(단수) → is.
04
Mailed-out coupons that — because of a printing error — offered no savings to recipients producing just as much customer response as did error-free coupons that offered substantial savings.
🔊
인쇄 오류 때문에 수령인에게 아무런 할인을 제공하지 않은, 우편으로 발송된 쿠폰이, producing ✗ — 29번 어법: 주절 본동사가 없어 비문 → produced(만들어 냈다)여야 함, 상당한 할인을 제공한 오류 없는 쿠폰만큼이나 많은 고객 반응을 만들어 냈다.
📌mailed-out우편으로 발송된printing error인쇄 오류savings할인(액), 절약recipient수령인, 받는 사람error-free오류 없는substantial상당한
✏️ 문법⚠ 어법 정답 포인트(분사 ↔ 본동사): 주어는 Mailed-out coupons (that ... offered no savings to recipients)이고, 그 뒤에 주절 본동사가 필요하다. 본문의 ✗ producing은 분사라 본동사 자리를 채우지 못해 문장이 비문이 됨 → ✓ produced(만들어 냈다, 과거 본동사). / as ~ as 도치: as much ... as did error-free coupons = 오류 없는 쿠폰이 (만들어 낸) 만큼(as 뒤 did + 주어 도치). / 주격 that: coupons that offered ....
EASY 쉬운 풀이 【실험 결과 + 어법】
인쇄 오류로 할인이 0원인 쿠폰이, 진짜 할인되는 쿠폰만큼 고객 반응을 끌어냈다. 사람들이 쿠폰을 보면 내용을 확인 안 하고 자동으로 반응한다는 증거. ★ 어법: 문장의 본동사는 produced. producing(분사)이면 동사가 없어 비문 → 29번 정답 포인트.
③
05
The obvious but instructive point here is that we expect discount coupons to do double duty.
🔊
여기서 분명하지만 시사하는 바가 큰 점은, 우리가 할인 쿠폰이 이중의 역할을 하기를 기대한다는 것이다.
📌obvious분명한instructive시사적인, 교훈적인expect O to VO가 ~하기를 기대하다do double duty이중 역할을 하다
06
Not only do we expect them to save us money, we also expect them to save us the time and mental energy required to think about how to do it.
🔊
우리는 쿠폰이 우리에게 돈을 절약해 주기를 기대할 뿐만 아니라, 그것(= 돈을 절약하는 일)을 어떻게 할지 생각하는 데 필요한 시간과 정신적 에너지도 절약해 주기를 기대한다.
📌not only ~~뿐만 아니라 (도치)save A BA에게 B를 아껴 주다mental energy정신적 에너지required to V~하는 데 필요한how to do it그것을 어떻게 할지
✏️ 문법부정어 도치 Not only: Not only do we expect ... — 부정어구 Not only가 문두 → 주어·동사 도치(do we expect). / 4형식 save A B: save us money / save us the time and mental energy = 우리에게 ~을 아껴 주다. / 과거분사 수식 required: the time ... required to think = 생각하는 데 필요한 시간(수동). / 의문사 + to V: how to do it.
EASY 쉬운 풀이 【이중 역할의 정체】
쿠폰은 ① 돈을 아껴 줄 뿐 아니라, ② "어떻게 아낄까"를 고민하는 시간·정신적 에너지까지 아껴 준다. 이것이 '이중 역할(double duty)'. ★ 핵심: 쿠폰의 두 번째 기능 = '생각의 수고' 절약.
07
In today's world, we need the first advantage to handle pocketbook strain; but we need the second advantage to handle something potentially more important — brain strain.
🔊
오늘날의 세상에서, 우리는 지갑(돈)의 부담을 처리하기 위해 첫 번째 이점이 필요하지만, 잠재적으로 더 중요한 무언가 — 즉 두뇌의 부담 — 를 처리하기 위해 두 번째 이점이 필요하다.
📌advantage이점, 장점handle처리하다, 다루다pocketbook strain지갑(돈)의 부담potentially잠재적으로brain strain두뇌의 부담
✏️ 문법목적의 to부정사: to handle pocketbook strain = 지갑 부담을 처리하기 위해. / -thing + 형용사 후치: something potentially more important = 잠재적으로 더 중요한 무언가(something은 형용사가 뒤에서 수식). / 대시(—) 동격: something ... important — brain strain(앞 something을 다시 명명).
EASY 쉬운 풀이 【결론 — 두 부담】
현대인에게 쿠폰의 첫째 이점은 지갑(돈) 부담을, 둘째 이점은 (어쩌면 더 중요한) 두뇌(생각) 부담을 덜어 준다. ★ 주제 = 쿠폰은 돈과 함께 '생각하는 수고'까지 줄여 준다.

문장 02 · 과거분사 offered / 계속적 which / allow O to V

Tier 2

문장 04 · ⚠ 본동사 producing → produced / as ~ as 도치
coupons that offered no savings ... producing just as much response as did error-free coupons.
할인 0원 쿠폰도 같은 만큼 반응을 끌어냈다.
Tier 1
⚠ 분사 ↔ 본동사 (29번 정답)
주어 Mailed-out coupons (that ... offered ...) 뒤에 주절 본동사가 필요. ✗ producing(분사) → ✓ produced(과거 본동사). producing이면 문장에 동사가 없어 비문.

문장 05 · expect O to V / do double duty
we expect discount coupons to do double duty.
쿠폰이 이중 역할을 하기를 기대한다.
Tier 1
expect + O + to부정사

Tier 3

obvious but instructive = 분명하지만 시사적인. do double duty = 이중 역할을 하다(한 번에 두 가지 기능).
문장 06 · 부정어 도치 Not only / 4형식 save A B / required

Tier 1

Tier 2
4형식 save A B / 과거분사 required
save us money / save us the time and mental energy(4형식). the time ... required to think = 생각하는 데 필요한 시간(과거분사 수동 수식). how to do it = 의문사+to V.
문장 07 · 목적의 to / -thing 후치 수식 / 대시 동격
