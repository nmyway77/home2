Self-Experiments First: The Optimal Research Strategy — 자기 실험 먼저: 최적의 연구 전략
2026년 3월 고2 모의고사 40번
🖱 PC: 클릭 → 해석 · 🔊 → 발음
📱 모바일: 탭 → 해석
💡 노란박스 → 쉬운 풀이
일반 문장
문법 핵심
01
When self-experimentation and conventional experiments are both possible, the difference between them often resembles the difference between learning and showing: Self experiments are better for discovery but worse for convincing others that the solution is helpful or the answer is correct.
🔊
자기 실험과 전통적 실험이 둘 다 가능할 때, 그 차이는 배움과 보여주기의 차이를 종종 닮는다: 자기 실험은 발견에 더 낫지만 다른 사람을 설득하기에는 더 나쁘다.
📌self-experimentation자기 실험conventional전통적인resembles닮다convincing설득하기solution해결책
✏️ 문법resembles = the difference = 단수 → resembles(단수 동사). better for ~ but worse for ~ = 비교급 병렬(대조). for convincing = 전치사 for + 동명사.
02
Of course, most scientists want to do both—discover something and convince others of their discovery.
🔊
물론, 대부분의 과학자는 둘 다 하길 원한다—무언가를 발견하고 자신의 발견에 대해 다른 사람을 설득하기.
📌discovery발견
✏️ 문법want to do = want + to V. discover and convince = 동사원형 병렬 (do의 내용). convince others of = ~에 대해 설득하다 (convince + O + of).
03
Thus, psychologists should consider doing both self-experiments and conventional ones.
🔊
따라서, 심리학자는 자기 실험과 전통적 실험 둘 다 하는 것을 고려해야 한다.
📌psychologist심리학자
✏️ 문법consider doing = consider + 동명사. ✗ consider to do → ✓ consider doing. ones = experiments 대용어.
04
The best use of resources may often be self-experiments followed by conventional ones.
🔊
자원의 최선의 사용은 자기 실험 후 전통적 실험이 뒤따르는 것일 수 있다.
📌resources자원
✏️ 문법followed by = ~가 뒤따르는 (과거분사 수식). experiments (which are) followed by ~.
05
The researcher begins with self-experiments that, if all goes well, find large effects and/or generate and eliminate many hypotheses.
🔊
연구자는 자기 실험으로 시작하는데, 순조로우면 큰 효과를 발견하거나 많은 가설을 생성하고 제거한다.
📌generate생성하다eliminate제거하다hypothesis가설
✏️ 문법that ~ find, generate, and eliminate = 관계대명사 that + 동사 3개 병렬. experiments = 복수 → find(복수). if all goes well = 삽입 조건절.
【핵심】 연구의 최적 전략! ①자기 실험(탐색·발견) → ②매개변수 최적화 → ③전통적 실험(검증·설득). 자기 실험 = 발견에 최적, 전통적 실험 = 설득에 최적. ★ 전자(자기실험)가 후자(전통실험)에 선행해야 가장 효율적!
06
This exploratory and theory-building phase lasts until a convenient solution or large effect is found.
🔊
이 탐색적이고 이론 구축적인 단계는 편리한 해결책이나 큰 효과가 발견될 때까지 지속된다.
📌exploratory탐색적인theory-building이론 구축적인phase단계convenient편리한
✏️ 문법lasts = phase = 단수 → lasts. until ~ is found = ~이 발견될 때까지 (until + 수동).
07
Then the researcher uses self-experiments to find the procedural parameters (e.g., duration, time of day, intensity) that optimize the solution or maximize the effect.
🔊
연구자는 해결책을 최적화하거나 효과를 극대화하는 절차적 매개변수(지속 시간, 시간대, 강도)를 찾기 위해 자기 실험을 사용한다.
📌procedural절차적인parameters매개변수duration지속 시간intensity강도maximize극대화하다
✏️ 문법to find = 목적의 to부정사. that optimize or maximize = 관계대명사 that (parameters 수식). parameters = 복수 → optimize(복수).
08
Only then would the researcher begin conventional experiments, using the optimized parameters.
🔊
그제서야 연구자는 최적화된 매개변수를 사용하여 전통적 실험을 시작할 것이다.
📌optimized최적화된
✏️ 문법Only then would ~ = Only + 부사 → 도치! would + S + V원형. 정상 어순: The researcher would begin only then. using ~ = 분사구문 (수단/동시). optimized = 과거분사 전치수식.
09
Since self-experiments are suited to exploration while conventional experiments provide validation, the most efficient strategy is for the former to precede the latter.
🔊
자기 실험이 탐색에 적합한 반면 전통적 실험이 검증을 제공하므로, 가장 효율적인 전략은 전자가 후자에 선행하는 것이다.
📌are suited to~에 적합하다strategy전략former전자precede선행하다
✏️ 문법are suited to = ~에 적합하다 (be suited to + 명사). for the former to precede = for + O + to V = O가 V하는 것 (의미상 주어). the former / the latter = 전자 / 후자.
📖 줄거리 & 핵심 정리
CORE MESSAGE
자기 실험(탐색) → 전통적 실험(검증) 순서가 가장 효율적!
= Self-experiments for discovery first, then conventional experiments for validation.

📐 글의 흐름 단계별로
1
차이 제시 — S01~S03
자기 실험 = 발견(learning). 전통적 실험 = 설득(showing). 둘 다 필요.
2
최적 순서 — S04~S07
자기 실험 먼저(탐색→가설→최적화) → 전통적 실험(최적화된 매개변수로).
3
결론 — S08~S09
Only then 전통적 실험 시작. 전자(자기)가 후자(전통)에 선행 = 가장 효율적.
🎯 수능 유형별 핵심 답
주제·요지자기 실험은 탐색에, 전통적 실험은 검증에 적합하므로 자기 실험이 전통적 실험에 선행하는 것이 가장 효율적이다

문장 01 · resembles 수일치 / for + 동명사
The difference resembles ~; better for discovery but worse for convincing.
차이가 닮는다; 발견에 낫지만 설득에 나쁨.
Tier 1
resembles = 단수 수일치
the difference resembles = difference = 단수 → resembles. ✗ resemble → ✓ resembles
Tier 2
for convincing = 전치사 + 동명사
worse for convincing = 설득하기에 더 나쁜. for + V-ing.
문장 02 · want to do / convince O of N / 동사원형 병렬
Scientists want to do both—discover and convince others of their discovery.
둘 다 하길 원한다—발견하고 설득하기.
Tier 2
discover and convince = 동사원형 병렬
do both—discover and convince = do의 내용을 구체화. 동사원형 병렬.
Tier 3
convince O of N = ~에 대해 설득
convince others of their discovery = 발견에 대해 설득. ✗ convince others about → ✓ convince others of
문장 03 · consider doing (동명사)
Psychologists should consider doing both.
둘 다 하는 것을 고려해야 한다.
Tier 1
consider + 동명사 = ~하는 것을 고려하다
consider doing = 하는 것을 고려. consider + V-ing. ✗ consider to do → ✓ consider doing
문장 04 · followed by (과거분사 수식)
Self-experiments followed by conventional ones.
자기 실험 후 전통적 실험.
Tier 2
followed by = ~가 뒤따르는 (과거분사)
followed by conventional ones = 전통적 실험이 뒤따르는. 과거분사 후치수식.
문장 05 · 관계대명사 that + 삽입절 + 동사 병렬
Experiments that, if all goes well, find, generate, and eliminate.
순조로우면 발견·생성·제거하는 실험.
Tier 1
that ~ find, generate, and eliminate = 관계절 + 동사 3병렬
experiments that find, generate, and eliminate = 관계대명사 that + 동사 3개 병렬. experiments = 복수 → find(복수). if all goes well = 삽입절.
문장 06 · lasts 수일치 / until + 수동
This phase lasts until a solution is found.
이 단계는 해결책이 발견될 때까지 지속.
Tier 1
lasts = phase 단수 수일치
this phase lasts = phase = 단수 → lasts. ✗ last → ✓ lasts
Tier 2
until ~ is found = ~이 발견될 때까지 (수동)
until a solution is found = 해결책이 발견될 때까지. until + 수동태.
문장 07 · to find (목적) / that optimize (관계절)
Uses experiments to find parameters that optimize or maximize.
최적화하는 매개변수를 찾기 위해 사용.
Tier 2
to find = 목적의 to부정사
uses experiments to find = 찾기 위해 사용. to V = 목적.
Tier 2
that optimize or maximize = 관계절 + 동사 병렬
parameters that optimize or maximize = 최적화하거나 극대화하는 매개변수. 복수 → optimize.
문장 08 · Only then would ~ (도치) / using (분사구문)
Only then would the researcher begin, using optimized parameters.
그제서야 시작할 것이다, 최적화된 매개변수 사용하여.
Tier 1
Only then would ~ = 부정어구 도치
Only then would the researcher begin = Only + 부사 → 도치(조동사 + S + V). 정상: The researcher would begin only then. ✗ Only then the researcher would → ✓ Only then would the researcher
Tier 2
using = 분사구문 (수단/동시)
using the optimized parameters = 최적화된 매개변수를 사용하여. 현재분사 분사구문.
문장 09 · be suited to / for O to V / the former·latter
Experiments are suited to exploration; strategy is for the former to precede the latter.
탐색에 적합; 전자가 후자에 선행하는 것.
Tier 1
be suited to + 명사 = ~에 적합하다
are suited to exploration = 탐색에 적합하다. suited to + 명사. ✗ suited for exploring → ✓ suited to exploration suited to = 적합한
Tier 1
for O to V = O가 V하는 것 (의미상 주어)
for the former to precede the latter = 전자가 후자에 선행하는 것. for + 의미상 주어 + to V.
Tier 3
the former / the latter = 전자 / 후자
the former = 전자(자기 실험). the latter = 후자(전통적 실험).
