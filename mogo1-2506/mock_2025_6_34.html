01
When scientists make an important new discovery or experimentally prove some hypothesis, they do not, in general, keep that information to themselves so that they alone can consider its meaning and derive additional theories from it. ★★★ not ~ so that ~ (부정 범위!)
🔊
과학자들이 중요한 새 발견을 하거나 어떤 가설을 실험으로 증명할 때, 그들(= 과학자들)은 일반적으로 그 정보(= 발견·가설)를 자기들에게만 간직하지는 않는다 — 자기들만이 의미를 고찰하고 그것으로부터 추가 이론을 끌어낼 수 있도록 하려는 것이 아니다.
📌discovery발견 / discover(동) → discovery(명) (어근군)experimentally실험적으로 / experiment(명·동) → experimentally(부)hypothesis가설 (복수: hypotheses)in general일반적으로keep A to oneselfA를 자신만 알고 있다·독점하다so that + S can V~할 수 있도록 (목적 부사절)derive A from BB로부터 A를 끌어내다·도출하다theory이론 (복수: theories)
✏️ 문법★★★ not ~ so that ~ (부정 범위 명확화 — v1.10!): do not keep ... so that they alone can derive = 그들만 끌어낼 수 있도록 간직하지는 않는다 — 부정은 "keep ~ so that 독점하려고" 전체에 걸림. 한국어 번역 시 부정어를 "간직하지는 않는다" 옆에 두어야 명확! ✗ "끌어낼 수 있도록 하지 않는다"(부정어 끝에 — 모호) → ✓ "간직하지는 않는다 — 끌어낼 수 있도록 하려는 것이 아니다"(대시 분리 + so that 의미 보존!). / ★★ keep A to oneself (~을 자신만 알다·독점): keep that information to themselves = 정보를 자기들끼리만 — 관용 표현. / ★ When 시간 부사절: When scientists make/prove ~ — when + S V. / ★ 부사 experimentally (실험적으로): experimentally prove — 동사 prove 수식. / ★ derive A from B (~에서 끌어내다): derive additional theories from it. / 동사 병렬 (and): consider its meaning and derive ~ — 두 동사원형 병렬.
【★★★ 핵심 어법 — 부정 범위 (not ~ so that ~) v1.10 신규!】 영어 어순(do not + 본동사 + so that 부속절)을 그대로 한국어로 옮기면 부정어가 끝까지 밀려 "무엇을 부정하는지 모호"해짐. 부정 범위 = "본동사 + so that 부속절" 전체에 걸림. 한국어 번역에서는 부정어를 본동사 옆에 두어 명확화! 처리 안 3가지: ① 대시 분리(본문 채택) ② 목적절 앞으로 ③ 두 문장 분리. ★ 부속절 자체의 의미(so that = ~할 수 있도록)도 반드시 한국어에 보존! ★ 시험 단골 — 의미 왜곡 위험! 부정문 + so that/because/to V 매번 점검!
02
Instead, they publish their results and make their data available for inspection. ★5형식 make + O + 형용사
🔊
대신, 그들(= 과학자들)은 자신들(= 과학자들)의 결과를 발표하고 자신들의 데이터를 검토에 이용할 수 있게 만든다.
📌instead대신 (단독 부사) / cf. instead of + 명사 (페어)publish발표하다·출판하다 / publication(명) (어근군)make A availableA를 이용 가능하게 만들다 (5형식)available이용 가능한 / avail(동) → available(형) (어근군)inspection검토·조사 / inspect(동) → inspection(명)
✏️ 문법★★ 5형식 make + O + 형용사 보어: make their data available = 데이터를 이용 가능하게 만들다 — V + O(data) + OC(available 형용사). ✗ make their data availably(부사 X) → ✓ make their data available(형용사). / ★ 동사 병렬 (and: publish + make): they publish ... and make ~ = 두 현재형 병렬. / ★ 접속부사 Instead (단독 부사): Instead, they publish = 대신 — 콤마 필수. cf. instead of + 명사 (페어 주의!) / ★ for + 명사 (목적·용도): for inspection = 검토를 위해 — 목적 표시.

03
This makes it possible for other scientists to reconsider their data and possibly refute their conclusions. ★★ 가목적어 it + 진목적어 to V
🔊
이것(= 데이터 공개)은 다른 과학자들이 그들(= 발표한 과학자들)의 데이터를 재검토하고 어쩌면 그들의 결론을 반박하는 것을 가능하게 만든다.
📌make it possible for + N + to VN이 ~하는 것을 가능하게 하다 (가목적어 + 진목적어)reconsider재검토하다 / re(다시) + consider (어원)possibly어쩌면·아마 / possible(형) (어근군)refute반박하다·논박하다 / refutation(명)conclusion결론 / conclude(동) → conclusion(명) (어근군)
✏️ 문법★★★ make it possible for + N + to V (가목적어 + 진목적어 + 의미상 주어): makes it possible for other scientists to reconsider = 다른 과학자들이 재검토하는 것을 가능하게 — it(가목적어) + possible(보어) + for + N(의미상 주어) + to V(진목적어). 5형식의 변형. / ★ 가목적어 it (긴 to V를 뒤로 미룸): 본래 make to reconsider possible이지만 to V 부분이 길어 it으로 대체 후 뒤로! / ★ 동사 병렬 (and: reconsider + refute): to reconsider ... and possibly refute = 두 동사원형 병렬 (둘 다 to V 의미 공유). / ★ 부사 possibly (어쩌면): possibly refute = 어쩌면 반박할 — 동사 수식.
【★★★ 핵심 어법 — make it possible for N to V (5형식 가목적어)】 S + make + it + 형용사 + for + N + to V: it은 가목적어(temporary), to V는 진목적어. for + N은 to V의 의미상 주어. cf. It is possible for N to V(가주어 버전)와 동족 패턴! ★ 시험 단골! 5형식 가목적어 — make/find/think/believe + it + 형 + to V!
04
More important, though, it makes it possible for other scientists to use that data to construct new hypotheses and perform new experiments. ★More important + 가목적어 + to V 목적
🔊
하지만, 더 중요한 것은, 그것(= 데이터 공개)이 다른 과학자들이 그 데이터를 사용해서 새로운 가설을 세우고 새로운 실험을 수행하는 것을 가능하게 만든다는 것이다.

✏️ 문법★★ make it possible for N to V (반복 패턴 — S03과 같음): makes it possible for other scientists to use ~ — 5형식 가목적어 + 의미상 주어. / ★★ to V 부사적 (목적): to use that data to construct new hypotheses = 데이터를 사용해서 가설을 세우기 위해 — 두 번째 to V는 목적. ✗ for constructing(다른 의미) → ✓ to construct(목적). / ★ 동사 병렬 (and: construct + perform): to construct ... and perform — 두 to V 병렬 (perform 앞 to 생략 가능). / ★ 접속부사 though (문중 — 그러나): More important, though, = 하지만 더 중요한 것은 — 콤마 사이 삽입. though = however의 격식체. / ★ 형용사구 More important (도치 강조): More important is that ~의 변형 — 강조용.
【★ 글의 흐름 강조 — More important, though】 S03에서 "재검토·반박"을 들고, S04에서 "더 중요한 것은"이라며 글의 핵심 효과를 도입: 데이터 공유가 새로운 가설·실험으로 이어진다는 것 (지식의 누적·발전). ★ 글의 핵심 주장 — 공유 → 새 가설·실험 → 지식 발전!

🌍 CORE MESSAGE — 글의 요지
과학자들은 발견을 독점하지 않고 결과·데이터를 공개한다.
그 덕에 다른 과학자들이 검증·반박하고, 더 중요하게는 새 가설·실험을 세운다.
정보가 소수에 갇히지 않고 널리 퍼질수록 사회 전체가 더 많이 알게 된다.
결론은 모든 과학자는 서로의 작업에 의존한다는 것.
🔑 핵심 — 공유가 지식을 키운다
✗ 독점: 발견을 자기만 간직 → 의미·이론을 혼자만 도출.
✓ 공유: 발표·데이터 공개 → ① 타인이 검증·반박 → ② (더 중요) 타인이 새 가설·실험 → ③ 정보가 널리 퍼질수록 사회 전체 지식↑ → ④ 과학자 간 상호 의존.
📐 글 전체 구조
행동
S1–2 · 독점 ✗ → 공개 ✓
과학자는 발견을 자기만 간직하지 않는다. 대신 결과를 발표하고 데이터를 검토에 개방한다.
효과
S3–4 · 검증 + 새 창조
공개 덕에 ① 남이 데이터를 재검토·반박할 수 있고, ② 더 중요하게는 새 가설을 세우고 새 실험을 수행할 수 있다.
가정
S5 · 널리 퍼질수록 더 안다
정보가 소수에 갇히지 않고 가능한 한 널리 퍼지면 사회 전체가 결국 더 많이 알게 된다는 전제.
결론
S6 · 상호 의존
엄밀히 말해 모든 과학자는 다른 과학자들의 작업에 의존한다.

Instead,(단독 부사 — 콤마 필수) ↔ instead of(전치사 + 명사) — 의미는 같으나 형태 다름. 시험 단골 페어!
Tier 3
for + N (목적·용도)
for inspection = 검토를 위해 — for + 명사 (목적·용도).
문장 03 · ★★★ make it possible for N to V (가목적어 + 의미상 주어 + 진목적어)
This makes it possible for other scientists to reconsider their data and possibly refute their conclusions.
이것(= 데이터 공개)은 다른 과학자들이 그들(= 발표한 과학자들)의 데이터를 재검토하고 결론을 반박하는 것을 가능하게 만든다.
Tier 1
★★★ make + it(가목적어) + 형용사 + for N + to V(진목적어)
makes it possible for other scientists to reconsider = 다른 과학자가 재검토하는 것을 가능하게 — it(가목적어) + possible(보어) + for + N(의미상 주어) + to V(진목적어). 본래 make to reconsider possible이 너무 길어 it으로 미룸.
Tier 1
★★ 가목적어 it 패턴 — 동족 동사군
make/find/think/believe/consider + it + 형/명 + to V 동족 패턴. 시험 단골! cf. It is possible for N to V(가주어 버전).
Tier 2
동사 병렬 (and: reconsider + refute)
to reconsider ... and possibly refute = 두 동사원형 병렬 (둘 다 to V 의미 공유 — to 생략).
Tier 3
refute / reconsider / conclusion (어휘)
refute(반박) / reconsider(re + consider — 재검토) / conclusion(conclude → conclusion 어근군).
문장 04 · ★ More important, though + 가목적어 + 부사적 to V 목적
More important, though, it makes it possible for other scientists to use that data to construct new hypotheses and perform new experiments.
하지만 더 중요한 것은, 그것(= 데이터 공개)이 다른 과학자들이 그 데이터를 사용해 새 가설을 세우고 새 실험을 수행하는 것을 가능하게 한다는 것이다.
Tier 1
★★ make it possible for N to V (S03 반복 패턴)
makes it possible for other scientists to use ~ — 5형식 가목적어 패턴 (S03과 동일).
Tier 1
★★ to V 부사적 (목적 — to construct)
to use that data to construct new hypotheses = 데이터를 사용해 가설을 세우기 위해 — 두 번째 to V는 부사적(목적). ✗ for constructing(다른 의미) → ✓ to construct(목적).
Tier 2
동사 병렬 (and: construct + perform)
to construct ... and perform ~ = 두 to V 병렬 (perform 앞 to 생략).
Tier 2
though (문중 — 그러나) + More important (강조)
More important, though, = 하지만 더 중요한 것은 — 콤마 사이 삽입. though = however 격식체. cf. More importantly(부사 — 동의 사항).
Tier 3
construct / hypotheses (복수) / experiment (어휘)
construct(세우다·구성) → construction(명). hypotheses(hypothesis 복수). experiment(명·동) → experimental → experimentally (어근군).
문장 05 · ★★ The assumption is that ~ + as~as possible + rather than V-ing + end up V-ing
The assumption is that society as a whole will end up knowing more if information is spread as widely as possible, rather than being limited to a few people.
그 가정(= 데이터 공개의 가정)은, 정보가 소수에게 제한되는 게 아니라 가능한 한 널리 퍼진다면, 사회 전체가 결국 더 많이 알게 된다는 것이다.
Tier 1
★★ The assumption is that ~ (보어 명사절)
The assumption is that society will end up knowing more = 가정은 ~라는 것 — that 절이 보어. ✗ The assumption is what ~(선행사 포함) → ✓ The assumption is that ~(접속사).
Tier 1
★★ rather than + V-ing (~보다는·~ 대신)
rather than being limited = 제한되는 것보다는 — rather than 뒤 V-ing(분사·동명사). ✗ rather than to be limited → ✓ rather than being limited. cf. instead of + V-ing 동족 의미.
Tier 1
★★ end up + V-ing (결국 ~하게 되다)
will end up knowing more = 결국 더 알게 될 것 — V-ing만! to V X. 동족: be busy V-ing, have fun V-ing, spend 시간 V-ing.
Tier 2
★ as + 부사 + as possible (가능한 한 ~하게)
as widely as possible = 가능한 한 널리 — 동족: as soon/much/often as possible.
Tier 2
if 조건절 (현재형 — 의미는 미래)
if information is spread = 정보가 퍼진다면 — if + 현재형. 주절 will과 호응.
Tier 3
as a whole / be limited to / spread (어휘)
as a whole(전체로서). be limited to(~로 제한 — 수동). spread(불규칙 — spread-spread-spread, 변하지 않음).
