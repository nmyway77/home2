01
All editing systems are now nonlinear computer-based systems that allow random access to any video shot or scene without having to fast forward or fast reverse to find it.
🔊
모든 편집 시스템은 이제, 어떤 영상 장면을 찾기 위해 빨리 감기나 빨리 되감기를 할 필요 없이 그 어떤 영상 컷이나 장면에도 무작위로 접근할 수 있게 해 주는 비선형 컴퓨터 기반 시스템이다.
📌editing system편집 시스템nonlinear비선형의random access무작위 접근without having to V~할 필요 없이fast forward빨리 감기
📐 문장 구조도
✏️ 문법주격 관계대명사 that: systems that allow ~ = ~을 가능하게 하는 시스템. that이 allow의 주어(선행사 systems). / without having to V: ~할 필요 없이. 전치사 without + 동명사(having) + to부정사. / to find it (목적): to find it = 그것을 찾기 위해.
【비선형 = ‘아무 데나 바로 접근’】 요즘 편집 시스템은 비선형(nonlinear) — 되감기·빨리감기 없이 원하는 장면에 바로 접근(random access)한다. 지문의 핵심 개념 제시. ★ random access = 무작위(즉시) 접근 → 비선형의 본질
02
Nonlinear systems can create a range of special effects, such as slow motion, wipes and dissolves.
🔊
비선형 시스템은 슬로 모션, 화면 전환(와이프), 디졸브 같은 다양한 특수 효과를 만들어 낼 수 있다.
📌create만들어 내다a range of ~다양한 ~special effects특수 효과such as ~~와 같은slow motion슬로 모션
📐 문장 구조도
✏️ 문법a range of ~: 다양한·여러 ~. / such as ~ (예시): ~와 같은 — 앞의 special effects를 예로 나열.
【다양한 특수 효과도 가능】 비선형은 접근성뿐 아니라 슬로 모션·와이프·디졸브 등 다양한 효과도 낸다. 비선형의 장점 추가. ★ such as = 예를 들어 → 효과의 구체 예시
03
Another highlight of a digital nonlinear system is its random access process that makes it easy for an editor to find desired shots or scenes without having to spend time fast forwarding or rewinding videotape.
🔊
디지털 비선형 시스템의 또 다른 장점은, 편집자가 비디오테이프를 빨리 감거나 되감는 데 시간을 들일 필요 없이 원하는 컷이나 장면을 쉽게 찾을 수 있게 해 주는 무작위 접근 방식이다.
📌highlight장점, 두드러진 점random access process무작위 접근 방식make it easy for O to VO가 ~하기 쉽게 하다spend time V-ing~하는 데 시간을 쓰다rewind되감다
📐 문장 구조도
✏️ 문법that makes + 가목적어 it ... to V: makes it easy for an editor to find ~ = 편집자가 ~을 찾기 쉽게 하다. it=가목적어, to find=진목적어, for an editor=의미상 주어. / 관계대명사 that: 선행사 process를 수식하는 주격 관계대명사. / without having to spend time V-ing: ~하는 데 시간을 쓸 필요 없이.
【핵심 장점 = 원하는 장면 ‘즉시’ 찾기】 또 다른 장점은 무작위 접근 — 테이프를 감을 필요 없이 원하는 장면을 바로 찾는다. S1의 random access를 편집자 관점에서 재강조. ★ make it easy for O to V = O가 ~하기 쉽게 하다(가목적어 it)
04
With nonlinear editing, shots or scenes can be easily added or removed anywhere in the program, and the computer adjusts the program length automatically.
🔊
비선형 편집에서는 컷이나 장면을 프로그램 어디에나 쉽게 추가하거나 제거할 수 있고, 컴퓨터가 프로그램 길이를 자동으로 조정한다.
📌add / remove추가하다 / 제거하다anywhere어디에나adjust조정하다automatically자동으로program length프로그램 길이
📐 문장 구조도
✏️ 문법수동태 can be added or removed: 추가되거나 제거될 수 있다. 조동사+be+과거분사(수동), 병렬. / and (등위접속): 앞뒤 두 절을 연결 — 자유 편집 + 자동 조정.
【추가·삭제 자유 + 길이 자동 조정】 비선형에선 장면을 어디든 자유롭게 넣고 빼며, 길이는 컴퓨터가 자동 조정. 편집 유연성의 핵심. ★ added or removed + adjusts automatically = 자유로운 편집
ⓑ
05
Linear editing was like composing a paper on a typewriter.
🔊
선형 편집은 타자기로 글을 작성하는 것과 같았다.
📌linear editing선형 편집be like V-ing~하는 것과 같다compose작성하다typewriter타자기
📐 문장 구조도
✏️ 문법be like + 동명사: was like composing ~ = ~하는 것과 같았다. like는 전치사(=~같은) → 동명사. / 과거시제 was: 과거의 방식(선형)을 설명.
【선형 = ‘타자기’로 글쓰기】 선형 편집을 타자기 글쓰기에 비유. 순서대로만 가능하고 수정이 어렵다는 함의(다음 문장). ★ like a typewriter = 타자기처럼 → 고치기 힘든 방식
06
If a mistake was made or new information needed to be added, the whole piece had to be retyped.
🔊
만약 실수가 있거나 새로운 정보를 추가해야 하면, 글 전체를 다시 타이핑해야 했다.
📌mistake실수need to be p.p.~되어야 하다the whole piece글 전체retype다시 타이핑하다had to V~해야 했다
📐 문장 구조도
✏️ 문법수동태 was made / had to be retyped: 실수가 만들어지면(수동) → 전체를 다시 쳐야 했다(had to be+p.p.). / need to be p.p.: needed to be added = 추가되어야 했다.
【실수 하나 = 전체 재작성】 타자기처럼 실수하면 처음부터 다시 쳐야 했다. 선형 편집의 결정적 단점. ★ the whole piece had to be retyped = 전체 재타이핑 → 선형의 비효율
ⓒ
07
Nonlinear editing, on the other hand, is like using a word processing program.
🔊
반면에, 비선형 편집은 워드 프로세싱 프로그램을 사용하는 것과 같다.
📌on the other hand반면에be like V-ing~하는 것과 같다word processing program워드 프로세서
📐 문장 구조도
✏️ 문법on the other hand (대조): 반면에 — 선형(타자기)과 비선형(워드)을 대조. / be like + 동명사: is like using ~ = ~을 사용하는 것과 같다.
【비선형 = ‘워드프로세서’】 반면 비선형은 워드프로세서에 비유 — 수정이 자유롭다는 함의(다음 문장). 타자기(선형)와 대조. ★ like a word processor ↔ typewriter : 수정 자유 vs 재작성
08
If a mistake is made, it is easily deleted and fixed with a few keystrokes, and new information can be added easily.
🔊
만약 실수가 생기면, 그것은 몇 번의 키 입력으로 쉽게 삭제되고 수정되며, 새로운 정보도 쉽게 추가될 수 있다.
📌delete삭제하다fix수정하다keystroke키 입력, 타건can be added추가될 수 있다a few몇 개의
📐 문장 구조도
✏️ 문법수동태 is made / is deleted and fixed / can be added: 실수가 생기면(수동) → 쉽게 삭제·수정되고(수동 병렬) → 추가될 수 있다(수동). / with a few keystrokes: 몇 번의 키 입력만으로.
【실수해도 몇 번 클릭으로 수정】 워드프로세서처럼 실수는 즉시 삭제·수정, 정보 추가도 쉽다. 비선형의 우수성으로 마무리(선형과 대조). ★ deleted and fixed with a few keystrokes ↔ retyped : 비선형의 편의성
📖 비선형 편집 vs 선형 편집 — 통합 요약
🎬 CORE MESSAGE — 전체 글의 메시지
비선형 편집 = 무작위 접근 + 자유로운 수정.
선형(타자기)은 순차적이라 수정 시 전체 재작성,
비선형(워드프로세서)은 몇 번의 키 입력으로 수정 끝
⚖️ 선형 vs 비선형
⌨️ 선형 (Linear)
타자기처럼 순차적. 실수·추가 시 전체를 다시 타이핑
= composing on a typewriter
VS
💻 비선형 (Nonlinear)
무작위 접근 + 특수효과. 어디든 추가·삭제·수정 자유, 길이 자동 조정
= using a word processor
📐 글 전체 흐름 — 단락 ⓐ·ⓑ·ⓒ
ⓐ
ⓐ — 비선형 시스템의 특징·장점
비선형은 되감기 없이 원하는 장면에 무작위 접근하고, 다양한 특수효과를 낸다. 편집자는 장면을 쉽게 찾고, 어디든 추가·삭제하며 길이는 자동 조정된다.
ⓑ
ⓑ — 선형 = 타자기 (비유)
선형 편집은 타자기로 글 쓰는 것과 같아, 실수하거나 정보를 더하려면 글 전체를 다시 타이핑해야 했다.
ⓒ
ⓒ — 비선형 = 워드프로세서 (비유)
반면 비선형은 워드프로세서와 같아, 실수는 몇 번의 키 입력으로 삭제·수정되고 정보 추가도 쉽다.
