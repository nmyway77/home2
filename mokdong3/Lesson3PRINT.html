관계부사
when·where·why·how
1. 관계부사의 역할: 관계부사절 안에서 시간·장소·이유·방법을 나타내는 ( )의 역할과 두 문장을 연결하는 ( )의 역할을 동시에 한다. ( )를 ( )하는 형용사절을 이끈다.
(ex) Do you know the place? + I can buy groceries at the place.
→ Do you know the place I can buy groceries at. (관계대명사 which 추가)
→ Do you know the place I can buy groceries. (전치사 at을 which 앞으로)
→ Do you know the place I can buy groceries. ('전치사+관계대명사'를 관계부사 where로)
2. 종류: 수식하는 ( )에 따라 다르게 쓰인다. 「 () + () 」로도 바꿔 쓸 수 있다.

가주어·진주어
It ~ to / that
1. 가주어, 진주어란?
• 형태: It ~ ( ) 또는 It ~ ( )
• 목적: 진주어를 문장 ( )로 보내고, 빈 자리에 가주어 ( )을 쓴다.
• 해석: “...하는 ( )은 ~다.” / 가주어 it은 [ 해석한다 / 해석하지 않는다 ]
진주어에 밑줄 후 해석해석
It is difficult to master cooking.요리를 마스터하는 ( )은 ( ).
It is dangerous to walk around alone at night.밤에 ( )은 ( ).
It is surprising that he was not the thief.그가 도둑이 아니었다는 ( ) ( ).
It is a fact that water boils at 100 degrees.
2. It ~ that 강조구문
문장에서 ( )하고 싶은 부분을 It과 that ( )에 넣는 형태로, “...한 것은 ( ) ~다”라고 해석한다.
Colin watched a soccer game last night. (Colin은 어젯밤 축구 경기를 봤다.)
It was ( ) that watched a soccer game last night.어젯밤 축구 경기를 본 사람은 바로 <>이다.
It was ( ) that Colin watched last night.Colin이 어젯밤 본 것은 바로 <>다.
It was ( ) that Colin watched a soccer game.Colin이 축구 경기를 본 것은 바로 <>이다.
3. 강조구문 vs 가주어·진주어 구분 Tip
① It·is·that을 지웠을 때 그 사이 표현이 들어갈 빈 자리가 있으면 ( ), 없으면 ( )이다.
② 진주어 구문 = that 이하 / 강조구문 = that 이하
1) It is Kimberly that teaches us math.( It~that 강조구문 / 가주어·진주어 )
2) It is true that she teaches us math.( It~that 강조구문 / 가주어·진주어 )
